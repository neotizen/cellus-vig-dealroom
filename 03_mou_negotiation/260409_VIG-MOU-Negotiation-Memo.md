## 배경 (Context)

셀루메드는 2026년 1월 티디랜드마크투자조합 1호(이하 "기존투자자")로부터 유상증자·구주 인수 거래를 진행하였으나, 감사의견 거절 및 상장폐지 위기로 인해 추가 구조적 해법이 필요해졌다. VIG Partners는 2026-03-19 에스테틱 플랫폼 투자 제안 및 텀시트 v2를 제출하였으며, 이에 기반해 MOU 체결을 통해 pre-pack M&A 구조를 확정하고자 한다.

### 당사자 현황
| 구분 | 당사자 | 비고 |
|------|--------|------|
| 셀루메드 | 대표이사 유인수 | 인스코비 대표이사 겸직 (이해충돌 검토 필요) |
| VIG Partners | 정연박 부대표 | 텀시트 v2 제출, IC 검토 중 |
| 기존 투자자 | 티디랜드마크투자조합 1호 (진광현/오힘찬) | 경영지배인 추천권 보유, 미국소송비용 집행 통제 중 |
| 인스코비 | 대표이사 유인수 | 셀루메드 구주 3,300,000주 보유 |

아래 각 조항별로 셀루메드 목표 포지션(Target), 수용 가능 범위(Acceptable), 절대 불가 선(Red Line)을 명시한다.

### 조항 1 — 거래 구조 (Deal Structure)

**쟁점:** VIG의 투자 방식 — 신주 인수(CB/신주), 구주 취득, 또는 에스테틱 사업부 직접 인수

| 구분 | 내용 |
|------|------|
| **VIG 예상 요구** | 에스테틱 carve-out 법인 신주 + 셀루메드 경영권 취득 (구주 포함) |
| **셀루메드 목표** | 에스테틱 사업부 물적분할 후 신설법인 신주 70% VIG 취득 + 셀루메드 존속법인 채무 정리 |
| **수용 가능** | 영업양수도 방식으로 에스테틱 자산 이전 후 신설법인 설립 |
| **RED LINE** | VIG가 셀루메드 부채(미국소송비용 140억, 인스그린 부채 등)를 무조건 승계하는 구조 거절 |

**협상 포인트:**
- Pre-pack 구조 상 법원 인가 전 거래 확정이 필요 → stalking horse bid 형태로 MOU 기재
- 기존 투자자(티디랜드마크)의 3,300,000주 구주 매각 여부 → VIG 구주 취득 시 기존 투자자 협의 선행 필수

### 조항 2 — 거래 가격 및 가격 조정 메커니즘 (Valuation & Price Adjustment)

**쟁점:** 에스테틱 사업부 가치 산정 기준 및 감사의견 거절에 따른 가격 변동 가능성

| 구분 | 내용 |
|------|------|
| **VIG 예상 요구** | 재무 DD 완료 후 가격 확정, 가격 조정 조항 폭넓게 요구 |
| **셀루메드 목표** | 에스테틱 사업부 EBITDA 기준 배수법으로 가격 사전 확정 (MOU 단계) |
| **수용 가능** | 가격 ±10% 범위 내 조정 허용 + 조정 트리거 사유 열거 방식으로 한정 |
| **RED LINE** | 상장폐지 확정을 자동 가격 재협상 트리거로 인정하는 조항 거절 |

**협상 포인트:**
- MOU에 "에스테틱 사업부 가격 산정 방식 합의 → Closing 조건 DD 완료 후 ±10% 범위 조정" 구조로 기재
- 상장폐지 시 시나리오를 MOU에 별도 명기: 상장 여부와 무관하게 가격 기준 유지 조항 확보
- 인스코비 구주(@3,000원 기준 체결) 관련: VIG 구주 취득 단가와 연동 여부 명확히 분리

### 조항 3 — 선행 조건 (Conditions Precedent)

**쟁점:** MOU 발효 및 본계약 체결을 위한 선행 조건 목록

| 구분 | 셀루메드 포지션 |
|------|----------------|
| **수용** | 이사회 결의 (셀루메드) |
| **수용** | 기존 투자자(티디랜드마크) 서면 동의 |
| **수용** | VIG 내부 IC 승인 |
| **수용** | 에스테틱 carve-out 기본 구조 합의 |
| **협상** | 감사의견 거절 해소 → "이의신청 제출 완료"로 대체 수용 가능 |
| **RED LINE** | 감사의견 거절 **완전 해소**를 선행조건으로 요구하는 경우 거절 (현실적으로 달성 불가) |
| **RED LINE** | 인스그린·환경이엔지 채무 **전액 정리**를 선행조건으로 요구하는 경우 거절 |

**협상 포인트:**
- 감사의견 거절 해소 대신 "거래소 이의신청 적시 제출 + VIG 통지" 조항으로 대체 제안
- 자회사 처리는 선행조건이 아닌 Closing condition 또는 covenant로 이전 요청

### 조항 4 — 독점 협상권 (Exclusivity)

**쟁점:** MOU 서명 후 VIG의 독점 협상 기간 부여 범위

| 구분 | 내용 |
|------|------|
| **VIG 예상 요구** | MOU 서명 후 90일 독점 협상 + 타 투자자 접촉 금지 |
| **셀루메드 목표** | 45일 독점 기간 + Pre-pack 법원 절차와 연동된 자동 연장 조항 |
| **수용 가능** | 60일 독점 + 30일 자동 연장 (양방 서면 동의 불필요) |
| **RED LINE** | 독점 기간 중 거래소 이의신청, 채권자 협의, 법원 절차 진행을 금지하는 조항 거절 |

### 조항 5 — Break-up Fee

**쟁점:** 어느 일방이 협상을 파기할 경우의 위약금 구조

| 구분 | 내용 |
|------|------|

| **수용 가능** | VIG 귀책 파기 시: 독점 기간 기회비용 정액 배상 3억원 |
| **RED LINE** | Break-up fee 무제한 또는 거래대금 대비 일정 비율(%) 방식 거절 |

### 조항 6 — 기존 투자자 계약과의 관계 정리 (Intercreditor / Contract Hierarchy)

**쟁점:** 티디랜드마크투자조합과 체결한 5개 계약(2026-03-27)의 효력과 VIG MOU의 우선순위

| 구분 | 내용 |
|------|------|
| **현황** | 경영지배인 추천권: 티디랜드마크 보유 / 미국소송비용 공동승인: 티디랜드마크 보유 / 인스코비 구주 3,300,000주: 인스코비 보유 |
| **셀루메드 목표** | VIG MOU에 "기존 투자자와의 계약 해소 또는 VIG 동의 조건부 효력 정지" 조항 삽입 |
| **수용 가능** | 기존 계약 권리의 VIG로의 양도 동의 조항 |
| **RED LINE** | 기존 계약 당사자(티디랜드마크, 인스코비)의 사전 서면 동의 없이 기존 계약 일방 해지 주장하는 구조 거절 |

**협상 포인트:**
- 대표이사 유인수의 셀루메드·인스코비 겸직 이해충돌 → 인스코비 구주 3,300,000주 처리 방향 VIG와 명확히 합의
- 미국소송비용 140억원 중 잔여 집행분(70억) 처리: VIG 딜 클로징 후 에스크로 또는 셀루메드 부담 구조 협의

### 조항 7 — 진술 및 보장 / 면책 (R&W / Indemnity)

**쟁점:** 감사의견 거절, 미공개 부채, 소송 등에 대한 R&W 범위

| 구분 | 셀루메드 포지션 |
|------|----------------|
| **수용** | 에스테틱 사업부 관련 자산·부채 R&W (carve-out 범위 내) |
| **수용** | 미국 소송은 별도 고지(Disclosure) 처리 — R&W 예외 항목으로 명기 |
| **수용** | 인스그린·환경이엔지 부채는 별도 Disclosure Schedule로 처리 |
| **협상** | 감사의견 거절 관련 재무 R&W: 에스테틱 carve-out 법인 범위로만 한정 |
| **RED LINE** | 셀루메드 전사 재무제표에 대한 무제한 R&W 거절 |
| **RED LINE** | APUS 관련 감사 이슈를 R&W 범위에 포함하는 조항 거절 |

**면책 상한(Cap):**
- 일반 R&W 위반: 거래대금의 10%
- 세금·환경·소송 관련 특정 사항: 거래대금의 20%
- 사기·고의 위반: 무제한 (표준)

### 조항 9 — Pre-pack 법원 절차 조항 (Court Process Clause)

**쟁점:** 기업회생 신청 전 VIG와의 사전 합의(pre-pack) 구조를 MOU에 어떻게 반영할지

| 구분 | 내용 |
|------|------|
| **셀루메드 목표** | MOU를 stalking horse bid의 사전 합의로 명기 + 법원 인가 후 본계약 자동 전환 |
| **수용 가능** | 법원 불인가 시 MOU 자동 해제 (셀루메드 귀책 없음으로 처리) |
| **RED LINE** | 법원 회생절차 개시 자체를 MOU 해제 사유로 규정하는 조항 거절 |

## 협상 전략 요약 (Negotiation Strategy)

### 레버리지 분석
| 레버리지 | 셀루메드 | VIG |
|----------|----------|-----|
| 에스테틱 사업부 가치 | 유일한 매력 자산 → 협상력 有 | Roll-up 전략상 필수 자산 |
| 시간 압박 | 상장폐지 기한, 이의신청 기한 | IC 승인 지연 부담 |
| 대안 투자자 | 없음 (현실적) | 에스테틱 플랫폼 대체 매물 탐색 가능 |
| 기존 계약 복잡성 | 기존 계약 해소 부담 | 기존 계약 정리 없이 딜 불가 |

### 우선 타결 항목 (Quick Win)
1. 독점 협상 기간 45~60일 합의
2. 감사의견 거절 → 이의신청 제출 완료로 선행조건 대체
3. R&W 범위를 에스테틱 carve-out 법인으로 한정

### 양보 순서
1. Break-up fee 셀루메드 부담분 소폭 상향 수용 가능 (최대 7억원)
2. 독점 기간 60일까지 수용 가능
3. DD 기간 중 재무 정보 추가 공개 범위 확대 가능

### 절대 고수 항목 (Non-negotiable)
- 상장폐지를 자동 가격 재협상 트리거로 인정 거절
- 셀루메드 전사 재무 R&W 거절
- 법원 회생절차 개시를 MOU 해제 사유로 인정 거절
